# gd32 串口中断收发数据



要实现的功能：

1. 以中断的方式接收数据
2. 以中断的方式发送数据

## 中断发送数据

用按键控制串口发送数据，用户按一下按钮，单片机发送一个数据。这里需要用到外部中断。

## 用串口助手发送数据，单片机接收数据

使用串口助手给单片机发送数据
